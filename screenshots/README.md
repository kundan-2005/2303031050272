# Screenshots for Submission

Place your captured screenshots here before final submission. Required screenshots:

1. Registration API request and response (POST to `/evaluation-service/register`).
2. Backend `/notifications` API request and response showing pagination/filter parameters.
3. Frontend Notifications page showing filters and sample notifications.
4. Any console logs showing logging middleware output (optional but recommended).

Name screenshot files descriptively, for example:
`1_registration_response.png`, `2_backend_notifications.png`, `3_frontend_ui.png`.
